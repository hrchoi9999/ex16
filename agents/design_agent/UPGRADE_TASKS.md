# Design Agent Upgrade Tasks

## 담당

업그레이드 기능의 PC UI/UX 설계, 이후 Mobile UI 분리 설계.

## 준비 작업

- 신규 기능별 화면 위치 정의
- 좌측 메뉴, 중앙 캘린더, 작업 영역 간 정보 배치 기준 관리
- 외부 일정 후보 검토/승인 UI 설계
- AI 실행 결과와 오류 메시지 UX 설계

## QA 기준

- 화면 요소가 Streamlit 상단 bar에 가려지지 않아야 한다.
- 주/일/월 view에서 정보 밀도와 조작 방식이 각각 적합해야 한다.
- 한글 텍스트가 버튼/카드/표 영역 밖으로 넘치지 않아야 한다.
