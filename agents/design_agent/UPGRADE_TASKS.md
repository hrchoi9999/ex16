# Design Agent Upgrade Tasks

## 담당

업그레이드 기능의 PC UI/UX 설계, 이후 Mobile UI 분리 설계.

## 준비 작업

- 좌측: 보기 선택, Google 계정/연동, 관심 사이트 수집 메뉴 배치
- 중앙: 월 기본 캘린더, 월 이동 꺾쇠, 일/주/월 캘린더 유지
- 우측: 일정 편집, Google 가져오기, 수집 후보 목록, 생산성/추천 영역 배치
- 하단: 선택된 날짜/일정 상세와 AI 채팅만 배치
- 외부 일정 후보 검토/승인 UI 설계
- AI 실행 결과와 오류 메시지 UX 설계

## QA 기준

- 화면 요소가 Streamlit 상단 bar에 가려지지 않아야 한다.
- 주/일/월 view에서 정보 밀도와 조작 방식이 각각 적합해야 한다.
- 한글 텍스트가 버튼/카드/표 영역 밖으로 넘치지 않아야 한다.

## 2026-06-19 병렬 수정 작업

- 3분할 PC 레이아웃의 좌/중/우 섹션 경계는 사용자가 마우스로 폭을 조절할 수 있게 한다.
- 월 캘린더의 요일, 날짜 숫자, 일정 pill은 과한 bold를 피하고 업무형 대시보드에 맞게 작고 차분하게 표시한다.
- 중앙 화면은 캘린더를 우선 표시하고, 메뉴/검색/AI 입력은 좌측 또는 우측 섹션으로 분리한다.
