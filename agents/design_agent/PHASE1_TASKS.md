# Design Agent Phase 1 Tasks

## 목표

`DESIGN_SPEC.md`의 디자인 프롬프트를 기준으로 Streamlit 앱을 현대적인 B2B 일정 관리 대시보드로 개선한다.

## 작업

- 디자인 토큰을 앱 스타일에 반영한다.
- 사이드바 내비게이션 구조를 설계한다.
- 상단 헤더 영역을 설계한다.
- 일정/태스크/데이터 위젯 레이아웃을 정리한다.
- 모바일 폭에서 내용이 겹치지 않는지 확인한다.

## 완료 기준

- UI가 네이비, 화이트, 라이트 그레이 중심 팔레트를 따른다.
- 일정 등록, 조회, 변경, 알림, 추천 기능이 첫 화면에서 찾기 쉽다.
- 디자인 변경 후 앱이 정상 실행된다.
