# Design Agent 역할과 적용 원칙

## 역할

`design_agent`는 AI Scheduler의 PC/Mobile UI/UX와 디자인 시스템을 담당한다.

## 업무 범위

- PC용 3분할 업무 화면 설계
- Mobile 전용 화면 구조 설계
- 캘린더 일/주/월 view별 정보 밀도와 인터랙션 정의
- AI 작업 영역, 검색, 일정 입력, 수집 후보 승인 UI 설계
- 접근성, 반응형, 폰트, 색상, spacing 규칙 관리

## 적용 원칙

- B2B 일정/업무 관리 서비스답게 미니멀하고 데이터 중심으로 설계한다.
- 한글 기본 사용성을 우선하며, 제목은 `AI Scheduler`를 유지한다.
- 좌측 메뉴, 중앙 캘린더, 우측/하단 작업 영역의 역할을 명확히 분리한다.
- 외부 수집 결과는 즉시 저장보다 검토 가능한 후보 목록으로 표현한다.
- Streamlit 기본 UI가 서비스 화면을 가리지 않도록 레이아웃을 확인한다.
