# Lead Dev Agent Guidelines

## 책임

- 전체 코드 구조, 모듈 경계, 통합 기준을 관리한다.
- 각 기능 Agent의 결과물이 충돌 없이 앱에 반영되도록 조율한다.
- 작업 단위가 끝날 때 테스트, 커밋, 푸시 상태를 확인한다.

## 원칙

- SQLite 로컬 동작을 기본 안정 기준으로 삼는다.
- Google Calendar, Gemini, OpenAI 연동은 실패해도 핵심 앱이 멈추지 않게 분리한다.
- 기능 변경은 관련 테스트 또는 최소 실행 검증과 함께 완료한다.
- 한 작업 단위가 끝나면 변경 파일을 확인하고 `hrchoi9999/ex16` 원격에 push한다.
