# Notification Agent 역할과 적용 원칙

## 역할

`notification_agent`는 중요한 일정 알림과 임박 일정 표시를 담당한다.

## 업무 범위

- 중요 일정과 마감 임박 일정 감지
- 일정 view별 알림 표시 정책
- 수집된 일정 후보 중 긴급 일정 강조
- reminder rule과 notification copy 관리

## 적용 원칙

- 알림은 중요도, 시작까지 남은 시간, 사용자의 확인 여부를 함께 고려한다.
- 알림 과잉을 피하고 실제 행동이 필요한 항목을 우선 표시한다.
- 외부 수집 일정은 사용자가 승인하기 전까지 후보 알림으로 분리한다.
