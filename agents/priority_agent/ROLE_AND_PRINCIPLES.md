# Priority Agent 역할과 적용 원칙

## 역할

`priority_agent`는 일정과 작업의 우선순위를 계산하고 추천 근거를 제공한다.

## 업무 범위

- 중요도, 마감 임박도, 소요 시간, 반복성 기반 점수화
- 오늘 확인해야 할 일정 추천
- productivity score와 주간 요약 지표 계산
- AI 추천 설명 문구 작성 기준 관리

## 적용 원칙

- 추천 점수는 사용자가 이해할 수 있는 기준으로 설명한다.
- 외부 수집 일정은 confidence와 출처 신뢰도를 함께 반영한다.
- 사용자가 직접 지정한 중요도는 자동 추천보다 우선한다.
