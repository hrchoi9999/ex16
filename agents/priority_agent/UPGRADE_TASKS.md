# Priority Agent Upgrade Tasks

## 담당

우선순위 추천, 일정 점수화, 생산성 지표 개선.

## 준비 작업

- 점수 기준: 중요도, 마감 임박도, 소요 시간, 외부 출처 confidence
- 추천 결과 설명 문구 정의
- 사용자 지정 우선순위와 AI 추천 우선순위 충돌 처리

## QA 기준

- 추천 근거가 짧고 이해 가능해야 한다.
- 사용자가 직접 설정한 중요도를 존중해야 한다.
- 외부 수집 일정은 확정 전까지 별도 후보 점수로 표시해야 한다.

## 2026-06-22 BabyAGI 우선순위 2 구현 작업

- 마감 리스크 예측 + 리마인드 코치를 `리스크 코치` 작업 메뉴로 추가한다.
- 일정별 위험도는 마감 임박도, 중요도, 실행 계획 완료율, 일정 충돌, 동기화 상태를 합산한다.
- 위험도 결과는 `안전`, `주의`, `위험` 배지와 점수, 사유, 다음 행동으로 표시한다.
- 리스크 계산 결과는 SQLite에 저장해 스캔 시점과 판단 근거를 추적한다.
