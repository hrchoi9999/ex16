# Collector Agent Upgrade Tasks

## 담당

외부 사이트/프로그램에서 일정 후보를 수집하는 기능.

## 준비 작업

- 서울50플러스 일자리몽땅 직업교육/취업지원 모집중 공고 후보 수집
- K-Startup 모집공고 모집중 사업 후보 수집
- 서버 시작 시 1회, 운영 중 3시간마다 수집 실행 기준 구현
- 수집 원문과 일정 후보 정규화 schema 정의
- 사용자 승인 전 후보 상태 유지

## QA 기준

- 수집 결과에는 출처와 수집 시각이 있어야 한다.
- 모호한 일정은 확정 저장하지 않아야 한다.
- 실패 시 사용자에게 원인과 다음 행동을 안내해야 한다.

## 2026-06-19 병렬 수정 작업

- 관심 사이트는 서울50플러스 일자리몽땅과 K-Startup 두 곳으로 제한한다.
- 서울50플러스는 직업교육, 취업지원 메뉴의 모집중 후보만 수집한다.
- K-Startup은 사업공고 > 모집중 페이지의 모집중 후보만 수집한다.
- 수집 후보는 바로 일정 등록하지 않고 우측 후보 목록에서 사용자가 선택한 항목만 일정으로 전환한다.
