# Collector Agent Upgrade Tasks

## 담당

외부 사이트/프로그램에서 일정 후보를 수집하는 기능.

## 준비 작업

- 서울50플러스 일자리몽땅 직업교육/취업지원 모집중 공고 후보 수집
- K-Startup 모집공고 모집중 사업 후보 수집
- 서버 시작 시 1회, 운영 중 3시간마다 수집 실행 기준 구현
- 수집 원문과 일정 후보 정규화 schema 정의
- 사용자 승인 전 후보 상태 유지

## QA 기준

- 수집 결과에는 출처와 수집 시각이 있어야 한다.
- 모호한 일정은 확정 저장하지 않아야 한다.
- 실패 시 사용자에게 원인과 다음 행동을 안내해야 한다.

## 2026-06-19 병렬 수정 작업

- 관심 사이트는 서울50플러스 일자리몽땅과 K-Startup 두 곳으로 제한한다.
- 서울50플러스는 직업교육, 취업지원 메뉴의 모집중 후보만 수집한다.
- K-Startup은 사업공고 > 모집중 페이지의 모집중 후보만 수집한다.
- 수집 후보는 바로 일정 등록하지 않고 우측 후보 목록에서 사용자가 선택한 항목만 일정으로 전환한다.

## 2026-06-22 지정 메뉴 수집 기준

- 서울50플러스 일자리몽땅 `직업교육/취업훈련`: `/vocationalTraining.do` 화면의 `in_appListAjax` API를 사용하고 `ANN_RCRT_STAT=IN17003` 신청/접수 가능 공고만 수집한다.
- 서울50플러스 일자리몽땅 `직업교육/AI디지털교육`: `education.do?cost=ALL&state=ALL&term=AI&type=KEYWORD` 목록에서 `AI디지털교육` 분류이며 `수강신청` 또는 `대기신청` 상태인 강좌만 수집한다.
- 서울50플러스 일자리몽땅 `취업지원/민간채용공고`: `bizSeUrl=IN49008&rcrtSeUrl=IN47002` 화면의 `in_appListAjax` API를 사용하고 `ANN_RCRT_STAT=IN17003` 공고만 수집한다.
- 서울경제진흥원 모집공고: `https://seoul.rnbd.kr/client/c030100/c030100_00.jsp`의 `모집중` 행만 수집한다.
- 서울창조경제혁신센터 알림마당/사업공고: 서울센터 공고 JSON과 오픈이노베이션 공고 페이지에서 제목 또는 기간상 마감일이 경과하지 않은 항목만 수집한다.
- K-Startup 모집공고: `bizpbanc-ongoing.do`의 모집중 항목을 수집하고, 요약 목록과 상세 목록이 함께 있는 HTML에서 필터명이 제목으로 오인되지 않도록 카드 제목을 우선 추출한다.
- 수집된 공고를 일정에 넣을 때는 모집기간의 종료일 또는 마감일을 기준으로 `[마감]` 일정을 생성하고, 원문 URL은 `source_url`과 상세 설명에 저장한다.
